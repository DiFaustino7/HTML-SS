Criando diferentes formas de Listas

Quando criado uma lista nao necessariamente precisa numerar. exemplo:


1- acordar 2- levantar 3- tomar cafe 4- escovar os dentes 5- arrumar a cama

Para criar uma lista da maneira certa na atualidade vamos utilizar a tag "OL" significa: Ordem List. com isso a nossa lista ficara facil de usar e facil tambem de se caso quisermos colocar um item no meio da lista por exemplo so adicionar onde quisermos como no seguinte exemplo:

  1. acordar
  2. tomar cafe
  3. escovar os dentes
  4. brincar com o cachorro
  5. ligar o computador para estudos

Observe que iniciamos com a Tag "ol" e dentro dela colocamos a tag "li" para agrupar nossa lista. facil e rapido.

Existem tipos de lista

Uma lista numerica depois do 1 é simples, basta criar uma tag dentro da "ol" a tag "type-1 Start=5, com isso nossa lista passa a ser uma lista numerica apartir do numero 5.

  1. acordar
  2. levantar
  3. tomar cafe
  4. escovar os dentes
  5. arrumar a cama

Para criar uma lista em ordem alfabetica iremos fazer o mesmo. Dentro da tag "ol" vamos colocar a tag "Type-A ou a" Para Letras Maiusculas ou Minusculas.

  1. acordar
  2. levantar
  3. tomar cafe
  4. escovar os dentes
  5. arrumar a cama

Para Criar uma lista em numeração romana é simples, Basta dentro da tag "ol" colocarmos a tag "Type-i ou I" para numeros romanos em grandes ou pequenos.

  1. acordar
  2. levantar
  3. tomar cafe
  4. escovar os dentes
  5. arrumar a cama
